# Payment Success Analyzer

A smart dashboard that analyzes payment transactions and provides insights to improve payment success rates.

## Features

- Payment success vs failure analysis
- Interactive charts
- Payment method distribution
- Transaction table
- Dark mode support
- Smart insights panel

## Tech Stack

Frontend:
- React
- Vite
- Recharts

Tools:
- Git
- GitHub
